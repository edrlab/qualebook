Meo:
  - "Lors de la première occurrence d'une abréviation ou d'un acronyme dans un ouvrage ou un chapitre, présenter sa forme développée pour en donner la signification. Cela peut se faire directement dans le texte, par exemple en écrivant « une DTD (déclaration de type de document) ». One can also link to a glossary page or using a tooltip that provides the definition. It is also possible to use the `<abbr>` element with the `title` attribute to indicate the meaning. This requirement strictly applies only to the first occurrence of the term."
Controle:
  - "Visually identify abbreviations and acronyms on each page. Verify that the full meaning is indicated upon their first appearance. The following methods are possible: directly in the text (e.g., in parentheses), via a link on the term pointing to its definition (e.g., in a glossary), or via the `<abbr>` HTML element with a populated `title` attribute.\n"
