Meo:
  - "Unless the content concerned is intended to be made visible and perceptible upon action by readers (tabs, drop-down menus, etc.): <ul><li>Do not use the display and visibility properties to hide the content.</li><li>Do not use the HTML hidden attribute to hide the content.</li><li>Do not give the content an ARIA aria-hidden true attribute.</li></ul> Use: <ul><li>CSS properties allowing the positioning of the content. content outside the browser's display area (position, text-indent) or to crop it (clip);</li><li>ARIA properties allowing a label to be associated with content (aria-label, aria-labelledby, aria-describedby);</li><li>or, in the case of a form field label, its title attribute.</li></ul>"
Controle:
  - "In managed code and CSS stylesheets of pages reviewed&nbsp;: <ul><li>Detect, using a code inspector content that would be hidden from display (besides those intended to be made visible on action of readers and readers).</li><li>Make sure that none of these content uses the techniques indicated in the implementation if they are intended to be rendered in a screen reader.</li></ul>"
