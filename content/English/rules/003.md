abstract: "Ebooks are divided into individual resources, representing specific portions of the work (page, chapter, etc.) called *Content Document*. Each of these resources is contained in a file and must contain a data in the source code that allows to write it or provide information about it. This reference relates to the metadata which, as its name indicates, allows to describe the page."
