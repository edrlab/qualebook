For blind people using assistive technology or for people wishing to read using synthetic voice features, it is necessary to know whether all content will be correctly rendered."

  - "In ONIX, use one or more of the following codes: code 52 from list 196; code 01 from list 81 and code 10 from list 81 as well as the codes corresponding to the specific contents if the latter are present in the book: code 14 from list 196; code 15 of list 196; code 16 of list 196; code V212 from list 175."
